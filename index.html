AfterClass

AfterClass is a social media site for MU students to connect and communicate with their peers. The site was built using a LAMP stack, and has functionality for account creation, login/logout, profile creation, creating/joining groups, and creating posts in a group. To allow for testing, the login screen inputs are prefilled with a test user's credentials.

Calorie Tracker

The calorie tracker is a work-in-progress web application that will allow users to track their calorie and nutrient intake, as well as their daily weight. The application uses a MERN stack. The front-end is partially complete, and a seperate GitHub repository exists with an in-progress REST API that will sync the front-end with a MongoDB database.
